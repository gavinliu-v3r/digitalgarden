### 2、快捷键

command + E : 模式切换 编辑/阅读  
command + option + E : 模式切换 源码/阅读  
ctrl+table ：窗口滑动（+shift 反转）

### 3、关系图谱

不支持复制 手动输入

颜色 HEX： `#EC0914` 红色  
表达式：tag:#App_System/index/First -file:tag_management  
颜色 HEX：#3981F3 蓝色  
表达式：tag:#App_System/index/Second -file:tag_management

插件：


下载方式：  
远程：[https://airtable.com/embed/shrdmp10Lxmf5Wmgl?backgroundColor=purple&viewControls=on](https://airtable.com/embed/shrdmp10Lxmf5Wmgl?backgroundColor=purple&viewControls=on)
