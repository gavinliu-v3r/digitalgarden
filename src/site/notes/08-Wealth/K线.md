## 1. K 线解读

![](/img/user/000_Obsidian/900_System/Assets/K%E7%BA%BF/file-20260404235140445.png)

### 1.1. 核心 K 线组合解读（趋势反转/延续信号，按**反转强度 + 常用度**排序）

所有组合均以**两日 K 线**为核心，核心看**实体覆盖关系、涨跌幅度、成交量**，红色=阳线（涨）、绿色=阴线（跌），先明确基础：

**阳线实体**：开盘价→收盘价的上涨区间；**阴线实体**：开盘价→收盘价的下跌区间；**影线**：上下波动的极值（参考性弱于实体）。

### 1.2. 一、基础反转组合（最常用，阴包阳/阳包阴核心）

|   |   |   |   |   |   |
|---|---|---|---|---|---|
|**组合名称**|**K 线形态**|**核心含义**|**出现位置**|**后续趋势概率**|**成交量要求**|
|**阳包阴**|次日阳线实体**完全覆盖**前一日阴线实体（开盘低于阴开，收盘高于阴收）|多头强势反攻，空头溃败|低位/下跌趋势末端/横盘突破|止跌上涨（反转）|放量（买盘资金确认）|
|**阴包阳**|次日阴线实体**完全覆盖**前一日阳线实体（开盘高于阳开，收盘低于阳收）|空头强势打压，多头失守|高位/上涨趋势末端/横盘破位|止涨下跌（反转）|放量（卖盘资金确认）|
|**阳包星**|次日阳线实体覆盖前一日**十字星/小阴小阳**（星线代表多空分歧）|多头打破分歧，启动拉升|低位/横盘|上涨（延续/反转）|温和放量即可|
|**阴包星**|次日阴线实体覆盖前一日**十字星/小阴小阳**|空头打破分歧，开始下跌|高位/横盘|下跌（延续/反转）|温和放量即可|

|   |   |   |   |   |
|---|---|---|---|---|
|**组合名称**|**K 线形态**|**核心含义**|**关键要求**|**后续趋势**|
|**曙光初现**|第一天大阴，第二天**低开高走**收中阳/大阳，阳线实体**深入阴线实体 1/2 以上**（未完全覆盖）|空头无力，多头开始反攻，底部预警|阳线实体越深、成交量越大，信号越强|大概率反弹（弱于阳包阴）|

|**早晨之星**|三日组合：大阴→十字星/小星线（跳空）→大阳，星线为多空分歧|经典底部反转，趋势大概率逆转|星线跳空、第三日阳线放量，信号有效|上涨|

### 1.5. 四、补充：强弱对比与核心使用原则

#### 1.5.1. 反转强度排序（从强到弱）

- 顶部下跌：**阴包阳 > 倾盆大雨 > 乌云压顶 > 黄昏之星**
- 底部上涨：**阳包阴 > 旭日东升 > 曙光初现 > 早晨之星**

#### 1.5.2. 必看 3 个关键（缺一不可，否则信号失效）

① **位置**：组合出现在**高位/低位**才是反转信号，在趋势中途可能只是震荡，比如上涨途中的乌云压顶可能是洗盘；

② **成交量**：反转组合必须**放量**，缩量的阴包阳/阳包阴大概率是假信号（资金未真正进场/出逃）；

③ **趋势**：顺大势而为，比如大熊市的阳包阴可能是反弹，而非反转，勿盲目抄底。

#### 1.5.3. 新手避坑

- 只看 K 线形态不看成交量=无效分析；
- 单根 K 线/单组组合不代表趋势，需结合**均线、筹码、大盘情绪**确认；
- 影线不参与“覆盖”判断，只看**实体部分**（比如阴包阳只看阴线实体是否完全盖过阳线实体，影线再长也没用）。

### 1.6. 五、一句话速记（方便盘中快速判断）

- 阳包阴，放量涨，低位买；阴包阳，放量跌，高位卖；
- 乌云压顶高开跌，曙光初现低开涨，实体过半才有效；
- 星线夹在涨跌中，放量破位是反转。
